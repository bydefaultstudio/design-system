Configuration

Create a config file with the following structure:

{
  "name": "my-project",
  "version": "1.0.0",
  "theme": "default"
}

Note: Make sure to update the theme field to match your project.

Available Options

Option Type Default Description
name string - Project name (required)
version string "1.0.0" Semantic version number
theme string "default" Theme identifier

Warning

Changing the name after deployment will break existing links.
