Body

Paragraphs

This is a default paragraph. Body type sets the rhythm of every long-form page on the site — articles, case studies, this styleguide. Sentences should breathe at this size and line-height without feeling cramped or flabby.

Two paragraphs in a row should sit with comfortable air between them. The space comes from the element’s own margin, never from a wrapper. If you find yourself adding gap on the parent to fix paragraph spacing, fix the paragraph instead.

.sidebar-intro — used under the centered logotype.

.sidebar-about-copy — short blurb used in the sidebar About block. Lighter weight, smaller size.

.post-excerpt — feed card secondary line. Faded color so the title carries the eye.

Links

Links

Inline body links use the system underline treatment — a default anchor sits inline with text and inherits underline color and offset from design-system.css. Hover state shifts the color toward --text-secondary.

External links auto-append a marker: an external link.

Visited links inherit the same color scale — no purple. Focus state shows a visible ring via :focus-visible.

Inline

Inline text elements

Use strong for emphasis that should survive copy-paste, and em for tonal stress. Use code for inline identifiers and Cmd+K for keys. Use mark sparingly to highlight.

Smaller helper text uses small. Strikethrough lives in s when something has been retracted.
